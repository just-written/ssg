# Static Site Generator!

Visit [the github page](https://www.github.com/just-written/ssg) for the README!

An SSG using what I already know from other tools instead of trying to learn something new.
Use golang template in .html files, .json files for loading data into them, and .md files for content.

[The github page](https://www.github.com/just-written/ssg) explains it better.
